# Downloads

These zip files are pre-packaged for installing CIT on **Claude Desktop** and **claude.ai** (browser).

If you are using **Claude Code** in a terminal, you do not need these zips. Go back to the root of the repo and follow the Claude Code instructions in [INSTALL.md](../INSTALL.md) instead.

---

## Which Files Do You Need?

| File | What It Does | Install First? |
|---|---|---|
| `cit-config.zip` | Shared config for all three skills | Yes, install this one first |
| `cit-chat.zip` | CIT for claude.ai conversations | After cit-config |
| `cit-code.zip` | CIT for coding and Claude Code | After cit-config |
| `cit-design.zip` | CIT for UI/UX design work | After cit-config |

You do not have to install all four. See [INSTALL.md](../INSTALL.md) for guidance on which skills suit your workflow.

---

## How to Install

1. Download the zip files you need (right-click and save, or click the file then Download)
2. Open Claude Desktop or go to claude.ai in your browser
3. Go to **Settings > Customize > Skills**
4. Click **Add skill** and upload `cit-config.zip` first
5. Repeat for each additional skill
6. Make sure each skill is toggled **on**
7. Start a new conversation and type `cit setup`

Full step-by-step instructions with troubleshooting for every OS are in [INSTALL.md](../INSTALL.md).
